Flex Wrap

O Flex Wrap define se os itens devem ou não quebrar a linha. Por padrão eles não quebram linha, ou seja , eles vem definidos como Row, isso faz com que os flex itens sejam compactados além do limite do conteúdo.

No Wrap

É o padrão, ou seja, não permite a quebra de linha, temos duas situações, uma onde os elementos se encaixam dentro do container, e outra eles começam a vazar.

Wrap

O wrap permite a quebra de linha assim que um dos flex itens não seja mais compactado dentro do container, passando o elemento para a linha de baixo.
Se o max-width do flex-container fosse aumentado para 400px os itens passam para a linha de cima, porém se for um tamanho que não comporta o item, exemplo 310 px, vamos ter uma sobra no container mas o item vai respeitar o container e se manter dentro dos limites dele e vai jogar o outro elemento para a linha de baixo.

Wrap-Reverse

Permite a quebra de linha assim que um dos flex itens não puder mais ser compactado, porém na direção contrária da linha acima.
