Flex-Flow

È um atalho para as propriedades flex-direction e flex-wrap.
Porém seu uso não é tão comum, visto que, quando mudamos o flex-direction para column, mantemos o padrão do flex-wrap que é nowrap. O ideal é usar ele para não termos que usar duas linhas para direcionar as propriedades, ou seja usamos uma linha para passar para o css duas propriedades de uma vez.

Default: Row Nowrap
